# ROCK

ROCK is a minimalist introspective game.

The player observes a rock.

There is no quest, no score, no narrator, no answer, no wisdom dispenser, and no explicit moral.

Rock exists.

## First build

- One rock
- One old dirty sidewalk
- Subtle ambient changes
- A tiny silent question box

## Run

Open `index.html` in a browser.
